УРОК 5. Математика

Языки программирования нередко используются для математических вычеслений: от простого сложения до высшей математики.
C# исключением не стал, и сегодня мы рассмотрим несколько способов работы с математикой.

Арифметика на кончиках пальцев

Как ты знаешь, в арифметике мы оперирует известными нам числами. И самые частые операции - это сложение, вычитание, умножение, деление, реже нахождение остатка.
Взглянем, как их можно применить:

int result1 = 1+2; // 3
int result2 = 1-2; // -1
int result3 = 1*2; // 2
double result4 = 1/2; // 0,5 (обязательно в таком случае нужно оздать дробную переменную)
int result5 = 1/2; // 0?!
int result6 = 3%2; // 1 ("%" находит остаток от деления, а не проценты)
int result7 = 1/0; // ошибка! DivideByZeroException - на ноль делить нельзя

Функции

Подключим к нашей математике и алгебру с геометрией. Там часто используются функции, по типу логарифма, синуса, косинуса и так далее.
Для работы с ними нужно использовать слово Math:

double rounded = Math.Round(0.6); // округление до целого (если округляется 0.5 - то округлится до ближайшего чётного - 0)
double ceiled = Math.Ceil(0.1); // округление до целого (если округляется 0.5 - то округлится до ближайшего чётного - 0)
double floored = Math.Floor(0.9); // округление "до пола": к ближайшему меньшему целому
double sin = Math.Sin(90); // синус 90°
